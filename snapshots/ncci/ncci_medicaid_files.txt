Medicaid Change Report Files The Medicaid Change Report files contain only the changes made in the
Medicaid NCCI edit files from the previous calendar quarter. Information on these files is contained
in Change Report File Information (PDF) . Medicaid NCCI Change Report (ZIP) Effective July 1, 2026;
Posted June 1, 2026 Medicaid NCCI Change Report (ZIP) Effective April 1, 2026; Posted March 01, 2026
Announcements Previous MUE Edits can be found in the Medicaid MUE Archive . Medicaid NCCI Edit Files
CMS posts the complete updated Medicaid National Correct Coding Initiative (NCCI) edit files for the
public on this page at the beginning of each calendar quarter. These replace the Medicaid NCCI edit
files from previous calendar quarters. The presence of an HCPCS/CPT code in a Procedure-to-Procedure
(PTP) edit - or a Medically Unlikely Edits (MUEs) value for an HCPCS/CPT code - doesn’t necessarily
indicate that the code is covered by any or all state Medicaid programs. Note: States cannot use the
files posted here for processing and paying Medicaid claims. States must download the NCCI edit
files available on the Regional Information Sharing System portal (RISSNET) and must ensure that
they or their vendor uses the appropriate Medicaid NCCI edits to adjudicate Medicaid claims. How to
Access the Files This page provides links to NCCI Edit files. To download each edit file, just
select the link listed below. After you select the desired link, the zip file should download. Each
zip file contains both text and Excel file(s) formats. Note: Excel limits the number of rows that
can be exported in a single file to 1,048,576 rows. The Practitioner and Outpatient Hospital PTP
edit files both contain significantly more rows than that. To download the complete edit file, save
the zip file to your computer and extract the text and Excel file(s). Related Downloads 2026 Quarter
